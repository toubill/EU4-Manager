# 🌍 Empire Universe 4 - Gestionnaire de Planètes v1

Outil de gestion pour les explorateurs d'Empire Universe 4. Permet d'enregistrer, rechercher et filtrer les planètes découvertes.

---

## 📥 Ajout de planètes

1. **Page "Aperçu"** → `CTRL+A` → `CTRL+C`
2. ![Aperçu 1](apercu1.png)
3. **Cliquer sur "Ajouter une planète"** → `CTRL+V`
4. ![Aperçu 2](apercu2.png)
5. **Page "Analyse de Ressource"** → `CTRL+A` → `CTRL+C`
6. ![Aperçu 3](apercu3.png)
7. **Dans la popup d'ajout de planete** → `CTRL+V`
8. ![Aperçu 4](apercu4.png)
9. **Cliquer sur "Ajouter"**

> ✅ Le CTRL+A massif fonctionne même avec le chat et les menus !

1. Télécharger `empire_manager.exe`

3. Lancer le fichier
4. Aucune installation nécessaire
